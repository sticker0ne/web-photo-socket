## Installation

```bash
$ npm install
```

## Configuration

In .env file you can config
- APP_PORT = 2500   - port on which application will be ran
- APP_ENABLE_CORS = true - enabling or disabling CORS

## Running the app

### Production mode

```bash
$ npm run build
$ npm run start:prod
```

### Development mode

```bash
$ npm run build
$ npm run start:dev
```
